## 12 Tools and Sources

### 12.1 Tools Used

The following tools supported the implementation and documentation of the project:

* **DeepLcom:** For translations from German into English.
* **Gemini:** As an AI assistant for Markdown formatting and code snippets.
* **Drawio:** For creating the diagrams.
* **Visual Studio:** As the development environment for programming.
* **GitHub:** For version control and publishing the code.
* **Siemens NX:** For the CAD work.
